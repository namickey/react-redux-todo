# react-redux-todo
todo app  

## demo
[https://namickey.github.io/react-redux-todo/](https://namickey.github.io/react-redux-todo/)  

## install
* nodejs、npm  
https://nodejs.org/ja/  
* redux、react-redux  
```
npm install --save redux react-redux
```

## create-react-app
```
npm install -g create-react-app
npm start
```
以下のURLでブラウザが自動起動する。  
http://localhost:3000  


## 公開用build
```
npm run build
```
buildディレクトリ配下をWEBサーバで公開する。
